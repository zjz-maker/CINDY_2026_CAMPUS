## 本地运行
1. 克隆仓库：`git clone https://github.com/[您的用户名]/star-dyne-recruitment-2026.git`
2. 在浏览器中打开 `index.html`

## 部署到GitHub Pages
1. 在GitHub仓库的Settings中
2. 找到"Pages"选项
3. 在"Source"中选择"main"分支
4. 保存后等待几分钟，网站即可访问
